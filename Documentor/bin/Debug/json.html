JSON

Working with JSON data.

Extract

Retieves the valud at a defined "path" into the JSON object.

Arguments

Pos Name Required Description
1 Path true A dot-delimited representation of the path into the object. Each property is defined by a segment. Numeric values prepended by a tilde ("~1") are interpreted as the index of an array. Example: "person.~2.name.first".

Samples

Input A JSON String
Command Json.Extract person.~1.name
Output The value of the "name" object of the second "person" object.
