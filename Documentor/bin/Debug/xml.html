XML

Working with XML strings.

Extract

Extracts a single value from an XML document parsed from the input string.

Arguments

Pos Name Required Description
1 XPath true The XPath identifying the desired XML node. The InnerText of the resulting node will be returned.

Samples

Input James Bond
Command Xml.Extract //name
Output James Bond

FormatNodes

Performs token replacement on each node of a specified XPAth and returns the concatenated result.

Arguments

Pos Name Required Description
2 Template true The template to apply to each node. XPath can be enclosed in brackets. These XPath expessions will be executed on the XML of that node and the resulting content will replace the token.
1 XPath true XPath to return a list of XML nodes.

Samples

Input <rows>
<row>
<name>James</name>
</row>
<row>
<name>Bond</name>
</row>
</rows>
Command Xml.FormatNodes //row "<p>Name: {name}</p>"
Output <p>Name: James</p><p>Name: Bond</p>

Transform

Transforms an XML document against an XSL stylesheet

Arguments

Pos Name Required Description
1 XSLT true The raw XSLT to transform the input string.
2 XML false The XML to transform. If not provided, the XML is formed from the active text.

Samples

Input An XML string
Command File.Read xslt-file.xslt => $xslt
Xml.Transform $xslt
Output The transformed XML
Input
Command File.Read xml-file.xml => $xml
File.Read xslt-file.xslt => $xslt
Xml.Transform $xslt $xml
Output The transformed XML
