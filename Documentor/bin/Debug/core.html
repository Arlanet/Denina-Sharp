Core

For working with the pipeline and its variables.

AppendVar

Appends to the value of a variable

Arguments

Pos Name Required Description
1 Variable Name true The name of the variable to which to append data.
2 Data false The value to append. If omitted, the input string will be appended.

Samples

Input
Command SetVar Name James
AppendVar Name " Bond"
ReadFrom Name
Output James Bond

Clear

Replaces the input text. (The same as ReplaceAll called with no arguments.)

Samples

Input The contents of War and Peace...
Command Clear
Output Absolutely nothing.

InitVar

Sets the value of a variable to an empty string. The variable can now be referenced without error.

Arguments

Pos Name Required Description
1 Variable Name true The name of the variable to set. Multiple variables can be specified. All will be initialized.

Samples

Input
Command InitVar Name Address City State Zip
Output None. The named variables are all initialized to empty strings.

Now

Returns the current date and time, formatted by an optional format string.

Arguments

Pos Name Required Description
1 Format String false The C# time format string with which to format the results.

Samples

Input
Command Now "ddd d MMM"
Output Wed 25 Feb

ReadFrom

Sets the active text to the contents of a variable.

Arguments

Pos Name Required Description
1 Variable Name true The name of the variable to be retrieved.

Samples

Input
Command SetVar Name "James Bond"
ReadFrom Name
Output James Bond

SetVar

Sets the value of a variable to the value provided. Does not change the input string.

Arguments

Pos Name Required Description
1 Variable Name true The name of the variable to set.
2 Value false The desired value. If not provided, the variable is set to an empty string (same as InitVar).

Samples

Input
Command SetVar Name "James Bond"
ReadFrom Name
Output James Bond

WriteTo

Writes the active text to the named variable.

Arguments

Pos Name Required Description
1 Variable Name true The name of the variable to which to write the input string.

Samples

Input James Bond
Command WriteTo Name
Output The variable "Name" now contains "James Bond".
