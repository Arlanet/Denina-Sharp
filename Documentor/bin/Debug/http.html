HTTP

Working with resources over HTTP.

Get

Makes an HTTP GET request and returns the result.

Arguments

Pos Name Required Description
1 URL false The URL to request. If not provided, the input string is assumed to be a URL.

Samples

Input http://denina.org
Command Http.Get
Output The contents of the page at denina.org
