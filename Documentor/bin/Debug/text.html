Text

For working with text.

Append

Appends a string to the input string.

Arguments

Pos Name Required Description
1 Suffix true The string to append to the input string.

Samples

Input James
Command Text.Append " Bond"
Output James Bond

ExtractRegex

Extracts a substring based on a supplied regex.

Arguments

Pos Name Required Description
1 Pattern true A regex string pattern. It will require an extraction expression.

Samples

Input James Bond
Command Text.ExtractRegex a(..)s
Output me

Format

Performs simple token replacement of variables and the active text.

Arguments

Pos Name Required Description
1 Format String true A format string suitable for usage in String.Format. The input string will replace the {0} token. Variable values will replace {variableName} tokens.

Samples

Input James Bond
Command Text.Format "My name is {0}."
Output My name is James Bond.
Input
Command SetVar Name "James Bond"
Text.Format "My name is {Name}"
Output My name is James Bond.

FormatLines

Performs a template formatting operation on every line in a string and concatenates the result.

Arguments

Pos Name Required Description
1 Format String true A format string suitable for usage in String.Format. Each line will replace the {0} token.

Samples

Input James Bond
Ernst Blofeld
Command Text.FormatLines <li>{0}</li>
Output <li>James Bond</li><li>Ernst Blofeld</li>

Prepend

Prepends a string to the input string.

Arguments

Pos Name Required Description
1 Prefix true The string to prepend to the input string.

Samples

Input Bond
Command Text.Append "James "
Output James Bond

Replace

Finds and replaces one string with another.

Arguments

Pos Name Required Description
2 New String false The new string. If this argument is not provided, String.Empty will be used (so, the Old String will simply be removed.)
1 Old String true The string to be replaced.

Samples

Input James Bond
Command Text.Replace "James "
Output Bond
Input James Bond
Command Text.Replace James "Bond. James"
Output Bond. James Bond

ReplaceAll

Completely replaces the input string.

Arguments

Pos Name Required Description
1 New String false The new string. After this filter executes, the active text will be set to this value. If no argument is provided, String.Empty is used, effectively clearing the active text.

Samples

Input Any
Command ReplaceAll "James Bond"
Output James Bond
Input Any
Command ReplaceAll
Output None

Trim

Trims whitespace from the ends of the input string.
