File

Working with files on the file system.

Read

Reads the content of a file on the file system.

Arguments

Pos Name Required Description
1 Path true The path to the file, relative to AppDomain.CurrentDomain.BaseDirectory. This value should not start with a leading slash as Path.Combine will interpret that as "root."

Samples

Input
Command File.Read my-file.txt
Output The contents of my-file.txt
