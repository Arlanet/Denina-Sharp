Denina Command Reference

Denina filters are grouped into categories.

  • CoreFor working with the pipeline and its variables.
  • FileWorking with files on the file system.
  • HTMLCreating and manipulating HTML strings.
  • HTTPWorking with resources over HTTP.
  • JSONWorking with JSON data.
  • SQLFor working with SQL databases.
  • TextFor working with text.
  • UrlWorking with URLs.
  • XMLWorking with XML strings.
