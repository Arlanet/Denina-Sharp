SQL

For working with SQL databases.

GetXml

Executes SQL against the specified connection string and returns an XML recordset.

Arguments

Pos Name Required Description
2 SQL false The SQL to execute. If omitted, the input string will be used.
1 Connection String Name true The name of a connection string key from the application configuration.

Samples

Input
Command Sql.GetXml myConnectionString "SELECT * FROM TableName"
Output An XML string
Input SELECT * FROM TableName
Command Sql.GetXml myConnectionString
