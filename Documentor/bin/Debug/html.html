HTML

Creating and manipulating HTML strings.

Extract

Extracts an element from an HTML string. (Relies on the HtmlAgilityPack.dll assembly, which must be available.)

Arguments

Pos Name Required Description
1 Path true The XPath(-ish) to the element, based on HtmlAgilityPack's language standard.

Samples

Input
James Bond
Command Html.Extract //div[@id='a']
Output James Bond

LineBreaks

Replaces line breaks with the corresponding HTML tag.

Samples

Input James
Bond
Command Html.LineBreaks
Output James<br/>Bond

Wrap

Wraps the input string in a specified HTML tag with optional class and/or ID.

Arguments

Pos Name Required Description
1 Tag Name true The name of the HTML tag in which to wrap the content.
2 Class false If provided, the tag will use this as a "class" attribute.
3 ID false If provided, the tag will use this as an "id" attribute.

Samples

Input James Bond
Command Html.Wrap div spy agent
Output <div id="spy" class="agent">James Bond</div>
